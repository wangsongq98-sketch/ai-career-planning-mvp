## 🚀 外部访问配置步骤

### 方案一：使用 Ngrok（最简单，推荐）

#### 步骤 1：下载 Ngrok

**方法 A：直接下载**
1. 打开浏览器访问：https://ngrok.com/download
2. 选择 Windows 版本下载（ngrok-v3-stable-windows-amd64.zip）
3. 解压到任意目录，例如：`C:\ngrok\`

**方法 B：使用浏览器下载**
```
https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-windows-amd64.zip
```

#### 步骤 2：注册 Ngrok 账号

1. 访问：https://dashboard.ngrok.com/signup
2. 使用邮箱注册（免费版即可）
3. 登录后，在左侧菜单找到 **Authtoken**
4. 复制您的 Authtoken（格式类似：`2xxxxxxxxxxxxxxxxxxxxxxx`）

#### 步骤 3：配置 Ngrok

打开命令提示符（CMD）或 PowerShell，运行以下命令：

```bash
# 进入 Ngrok 目录
cd C:\ngrok

# 配置 Authtoken（将 YOUR_TOKEN 替换为您复制的 token）
ngrok config add-authtoken YOUR_TOKEN
```

#### 步骤 4：启动 Ngrok 隧道

在 Ngrok 目录下运行：

```bash
ngrok http 3000
```

#### 步骤 5：获取公网链接

运行后，Ngrok 会显示类似以下内容：

```
Session Status                online
Account                       your-email@example.com
Version                       3.x.x
Region                        United States (us)
Web Interface                 http://127.0.0.1:4040
Forwarding                    https://xxxx-xxxx-xxxx.ngrok-free.app -> http://localhost:3000

Connections                   ttl     opn     rt1     rt5     p50     p90
                              0       0       0.00    0.00    0.00    0.00
```

**您的公网访问链接就是**：
```
https://xxxx-xxxx-xxxx.ngrok-free.app
```

#### 步骤 6：分享链接

将这个公网链接分享给外部开发人员，他们可以通过这个链接访问您的项目。

**访问路径**：

### 方案二：使用 Cloudflare Tunnel（更稳定）

#### 步骤 1：安装 Cloudflare Tunnel

打开 PowerShell（以管理员身份运行）：

```bash
winget install --id Cloudflare.Cloudflare -e
```

#### 步骤 2：登录并创建 Tunnel

```bash
# 登录 Cloudflare
cloudflared tunnel login

# 创建 Tunnel
cloudflared tunnel create ai-career-mvp

# 配置 DNS（可选，需要有域名）
cloudflared tunnel route dns ai-career-mvp ai-career-mvp.your-domain.com
```

#### 步骤 3：启动 Tunnel

```bash
cloudflared tunnel run ai-career-mvp
```

#### 步骤 4：获取访问链接

Cloudflare 会提供一个子域名，例如：
```
https://ai-career-mvp.your-domain.com
```

### 方案三：使用花生壳（国内访问快）

#### 步骤 1：下载花生壳

访问：https://hsk.oray.com/download/

下载 Windows 版本并安装

#### 步骤 2：注册账号

打开花生壳，使用手机号注册账号（免费版即可）

#### 步骤 3：配置端口映射

1. 在花生壳界面点击"端口映射"
2. 点击"添加映射"
3. 配置如下：
   - 主机端口：3000
   - 内网 IP：127.0.0.1
   - 内网端口：3000
   - 类型：HTTP
   - 名称：AI职业规划一体机
4. 点击"保存"

#### 步骤 4：获取访问链接

花生壳会提供一个域名，例如：
```
http://yourname.orayddns.com:3000
```

## ⚠️ 常见问题

### 1. Ngrok 启动失败
**问题**：`Error: port 3000 is already in use`

**解决**：确保开发服务器正在运行，或者修改端口
```bash
ngrok http 3000 --host-header=localhost:3000
```

### 2. 防火墙阻止访问
**解决**：在 Windows 防火墙中允许 node.exe 和 ngrok.exe 通过防火墙

### 3. 局域网无法访问
**解决**：确保使用 `--host` 参数启动开发服务器
```bash
npm run dev -- --host
```

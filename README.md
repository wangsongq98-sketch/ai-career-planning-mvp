# AI职业规划一体机 MVP 原型

## 项目说明

这是一个基于 React + TypeScript 的可交互 Web 原型，包含学生端和管理平台两部分，支持多学校SaaS模式。

## 功能特性

### 学生端（竖屏，1080×1920）
- **注册页**：支持学校、院系、班级的级联选择
- **登录页**：支持学号密码登录和"立即体验"功能
- **首页**：显示学生信息、统计数据和最近面试记录
- **面试答题页**：支持录音、AI分析和评分
- **面试报告页**：显示综合得分和各维度评分
- **个人中心**：显示学生信息和退出登录

### 管理平台（宽屏，1440×900）
- **登录页**：支持学校管理员和超级管理员登录
- **学校管理**（超级管理员）：管理所有学校
- **院系管理**：按学校查看和管理院系
- **班级管理**：按院系查看和管理班级
- **学生管理**：查看和管理学生信息
- **仪表盘**：显示关键指标和趋势图表
- **面试记录列表**：查看所有面试记录
- **数据统计页**：显示各专业平均分和得分分布

## 技术栈

- React 18
- TypeScript
- React Router 6
- Tailwind CSS
- Lucide React Icons

## 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 模拟数据

所有数据使用硬编码 JSON 模拟，包括：
- 学校数据（南京大学、东南大学）
- 院系数据
- 班级数据
- 学生数据
- 岗位数据
- 面试题库
- 面试记录

## 路由说明

### 学生端路由
- `/student/register` - 注册页
- `/student/login` - 登录页
- `/student` - 首页
- `/student/interview` - 面试答题页
- `/student/report` - 面试报告页
- `/student/profile` - 个人中心

### 管理端路由
- `/admin/login` - 登录页
- `/admin/schools` - 学校管理
- `/admin/departments` - 院系管理
- `/admin/classes` - 班级管理
- `/admin/students` - 学生管理
- `/admin/dashboard` - 仪表盘
- `/admin/interviews` - 面试记录列表
- `/admin/statistics` - 数据统计

## 使用说明

1. **学生端体验**：
   - 直接访问 `/student/login` 页面
   - 点击"立即体验"按钮，使用预设学生"王小明"登录
   - 或注册新学生账号

2. **管理端体验**：
   - 访问 `/admin/login` 页面
   - 使用以下账号登录：
     - 学校管理员：`admin_nju` / `123456`（南京大学）
     - 学校管理员：`admin_seu` / `123456`（东南大学）
     - 超级管理员：`super` / `123456`（可切换学校）

## 注意事项

- 本项目仅用于演示和原型验证
